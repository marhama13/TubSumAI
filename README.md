# TubSumAI
TubeSumAI — A Streamlit web app that summarizes any YouTube video using its transcript and a locally-running Llama 3.2 model via Ollama. 100% private, no API keys, no cloud calls.

TubeSumAI is a simple, privacy-first web app built with Streamlit that turns any YouTube video into a structured AI summary. Just paste a video URL, and the app fetches the transcript via the YouTube Transcript API, then runs it through Llama 3.2 (via Ollama) running locally on your machine to generate an executive summary, key takeaways, insights, and action items.
Features

Paste any YouTube URL (videos, tutorials, podcasts, talks)
Automatic transcript extraction
Local AI summarization using Llama 3.2 via Ollama — no API keys, no data sent to the cloud
Clean, modern UI with a custom light theme
Download summaries as a text file
